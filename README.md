# Sistema de Cadastro de Usuários

Projeto desenvolvido em Python com interface gráfica utilizando Tkinter,
com o objetivo de praticar conceitos básicos de Análise e Desenvolvimento de Sistemas.

## Funcionalidades
- Cadastro de usuários
- Exibição de usuários cadastrados
- Interface gráfica simples

## Tecnologias
- Python
- Tkinter

Projeto acadêmico para fins de aprendizado.
